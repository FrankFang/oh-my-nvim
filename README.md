| --------------------------------------------- | ---------------- | ------ | -------------- |

| <kbd>leader</kbd> <kbd>f</kbd> <kbd>f</kbd>   | Search Files     |        |                |

| <kbd>leader</kbd> <kbd>f</kbd> <kbd>b</kbd>   | Search Buffers   |        |                |
